# Redovisningsbyra Site

Produktionsinriktad grund for en foretagshemsida byggd med:

- Next.js (App Router)
- React
- TypeScript
- Tailwind CSS
- ESLint
- Prettier

Ingen faktisk hemsida ar byggd an, endast miljo och struktur.

## Kom igang

```bash
npm install
npm run dev
```

## Scripts

```bash
npm run dev
npm run build
npm run start
npm run lint
npm run typecheck
npm run format
npm run format:check
```

## Struktur

```text
src/
  app/
  components/
    layout/
    sections/
    ui/
  config/
  content/
  lib/
  styles/
  types/
```
